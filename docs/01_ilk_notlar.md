# İlk Notlar

İlk hafta temel hedef, projenin düzenli bir GitHub reposu olarak başlatılmasıdır.

## Yapılacaklar

- README dosyası hazırlanacak
- Proje fikri yazılacak
- Python paket klasörü oluşturulacak
- Küçük örnek metin dosyaları kullanılacak
- Ham veri ve özel dokümanlar GitHub'a yüklenmeyecek

## İlk Teknik Yaklaşım

Proje ilk başta basit tutulacaktır:

1. Örnek metin dosyasını oku
2. Metni temizle
3. Metni parçalara ayır
4. Basit arama yap
5. Sonuçları kullanıcıya göster

Bu yapı daha sonra RAG sisteminin temelini oluşturacaktır.
