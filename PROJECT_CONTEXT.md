# Proje Bağlamı

## Proje Adı
Python Trading RAG Assistant

## Amaç
Trading eğitim notları, strateji açıklamaları ve örnek transkriptler üzerinde çalışabilecek Python tabanlı bir doküman analiz sistemi geliştirmek.

İlk sürümde sistem:
- Metin dosyalarını okuyacak
- Metni temizleyecek
- Metni yönetilebilir parçalara ayıracak
- İlerleyen aşamada bu parçalar üzerinde arama ve RAG işlemleri yapacak

## Kapsam Dışı
- Canlı al-sat işlemi
- Broker API bağlantısı
- Gerçek para ile işlem
- Yatırım tavsiyesi
- Otomatik emir gönderimi
